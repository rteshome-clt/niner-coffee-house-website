# Niner Coffee House

A responsive coffee shop website built with HTML and CSS featuring a menu, location map, and contact information.

## Project Overview
Niner Coffee House is a static website created as part of a web development course. The site presents a fictional coffee shop and includes sections for an about description, menu items, location and hours, and contact details. The project focuses on clean layout, semantic HTML, and custom CSS styling.

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (custom properties, flexbox, responsive layout)
- Google Fonts (Open Sans, Oswald)
- Font Awesome icons
- Google Maps Embed

## Features
- Multi-section layout (About, Menu, Location & Hours)
- Styled menu with prices using Flexbox
- Custom color palette using CSS variables
- Background image with overlay for readability
- Embedded Google Map for location
- Clickable phone and email links
- Responsive design for different screen sizes

## What I Learned
- Writing semantic HTML for accessibility and structure
- Using CSS variables for consistent theming
- Applying Flexbox for layout alignment
- Integrating external libraries and embeds
- Organizing files for a real-world web project

## How to View the Project
1. Clone the repository
2. Open `index.html` in a web browser  
   **or**
3. Use a live server extension (e.g., VS Code Live Server)
